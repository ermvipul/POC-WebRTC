## Overview
The signaling server is responsible for facilitating the communication setup for WebRTC (Web Real-Time Communication) by exchanging signaling messages. These messages include information about the media configuration, network information, and other metadata required to establish a direct peer-to-peer connection.

## Installation
Follow these steps to install and set up the project locally.

1. **Install dependencies:**
    ```sh
    npm install --force (frontend)
    npm install express socket.io cors (signaling)
    ```

2. **Run the app**
    ```sh
    npm run dev (frontend)
    node signaling_server.js (signaling)
    ```
